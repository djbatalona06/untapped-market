# Legacy archive

`archive/untapped-market-v1.html` is a **frozen historical snapshot** of the original
single-file Untapped Market site (React via CDN + Babel-in-browser, ~6 strains / 4
dispensaries, map placeholder).

It is **not deployed and not maintained**. The live product is the Vite + React + TS
app under `/src`, built with `npm run build` and deployed from `dist/` (see
`.github/workflows/deploy.yml`).

Do not edit this file or port logic out of it — it exists only as a reference of where
the project started. All new work happens in the `src/` application.
